# Personal Finance App

A personal finance app to track income, expenses, and savings.  
Built with *HTML, CSS, JavaScript, Bootstrap, and **LocalStorage*.

---

## Features

- Sign-up / Login  
- Light & Dark Mode Dashboard  
- Add Income & Expenses  
- Transactions List  
- Reports & Charts  
- Export Data to CSV/Excel
